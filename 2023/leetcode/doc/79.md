# LeetCode 每日一题 Daily Challenge  79 Word Search

给定一个二维网格和一个单词，找出该单词是否存在于网格中。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。





### 二、思路

> 第一感觉是什么，放弃奇淫技巧。



- 数据结构：二维数组
- 方法：深度优先搜索
-  深度：word长度，相同字母防止死循环。
